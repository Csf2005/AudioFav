escrição das Classes
-Audio

Classe base que representa um conteúdo de áudio genérico.

Responsabilidades:

Armazenar informações comuns como título e duração

Servir como superclasse para outros tipos de áudio

-Musica

Classe que herda de Audio e representa uma música.

Atributos adicionais:

Artista

Funcionalidades:

Define sua própria forma de classificação

-Podcast

Classe que herda de Audio e representa um podcast.

Atributos adicionais:

Apresentador

Funcionalidades:

Define sua própria forma de classificação

-MinhasFavoritas

Classe responsável por avaliar os áudios e indicar quais são considerados favoritos, com base em critérios de classificação.

-Main

Classe principal do projeto.

Função:

Instanciar objetos de música e podcast

Executar os métodos

Demonstrar o funcionamento do sistema
